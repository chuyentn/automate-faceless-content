# ✅ YouTube Optimization Checklist

Use this checklist for every video you publish to maximize views and growth.

---

## Before Publishing

- [ ] **Title**: Contains main keyword + is compelling (under 60 characters)
- [ ] **Thumbnail**: Eye-catching, readable text, contrasting colors
- [ ] **Description**: First 2 lines contain hook + keyword
- [ ] **Tags**: 10-15 relevant tags including long-tail keywords
- [ ] **Category**: Correct category selected
- [ ] **End Screen**: Added with subscribe button + next video
- [ ] **Cards**: Added at key moments in the video
- [ ] **Playlist**: Added to relevant playlist

## After Publishing

- [ ] **Pin a comment** with CTA or affiliate link
- [ ] **Share** to other platforms (TikTok, Instagram, Facebook)
- [ ] **Respond** to early comments within first hour
- [ ] **Check analytics** after 24 hours
- [ ] **Adjust** future content based on performance

## Weekly Review

- [ ] Check which videos performed best
- [ ] Analyze audience retention graphs
- [ ] Review traffic sources
- [ ] Update content strategy based on data
- [ ] Plan next week's content batch
